## 🚀 The Problem forkfind Solves

Ever been in this situation?

> *"Let's go out tonight!"*
> *"Sure! Where?"*
> *"...I don't know."*

Whether it's a **sudden plan**, a **family occasion**, or a **party with friends** — deciding where to eat is always harder than it should be. Existing apps focus on **food delivery**, not on the experience of **going out and dining together**.

**forkfind** is built for the people who want to:
- 🥂 Celebrate special moments at the right place
- 👨‍👩‍👧 Take their family somewhere everyone will love
- 💸 Find great food that fits their actual budget
- 🗺️ Get there without any hassle

## 🛠️ Tech Stack

forkfind is built entirely with **free, open-source tools** — no paid APIs, no credit card required.

```
Frontend      →  HTML5, CSS3, Vanilla JavaScript
Map Engine    →  Leaflet.js + OpenStreetMap
Restaurant Data →  Overpass API (OpenStreetMap)
Geocoding     →  Nominatim (free reverse geocoding)
Directions    →  OpenStreetMap Directions (OSRM)
Fonts         →  DM Sans + Playfair Display (Google Fonts)
Hosting       →  Netlify (free tier)
```

## 🏁 Getting Started

No installation. No setup. No API key. Just open and use.

### Option 1 — Use the live app
👉 **[https://forkfind.netlify.app/](https://forkfind.netlify.app/)** *(replace with your actual link)*

### Option 2 — Run locally

```bash

# 2. Open the folder
```

## 🗺️ How It Works

```
User opens app
      ↓
App detects live location (GPS)
      ↓
Overpass API fetches real restaurants nearby
      ↓
User applies filters (Budget · Diet · Cuisine · Open Now)
      ↓
Restaurants shown as cards + map pins in real time
      ↓
User taps a restaurant → sees details on map
      ↓
Taps "Get Directions" → live route opens instantly
```
